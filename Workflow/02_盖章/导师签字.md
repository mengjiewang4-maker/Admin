# 导师签字

> 几乎所有行政事务的第一步。

---

## 流程

```
微信问导师是否方便 → 方便 → 当面签
                    → 不方便 → 电子签 / 签字章 / 代签
```

---

## 三种替代方式

| 方式 | 什么时候用 |
|------|-----------|
| 电子签 | 导师远程，文件可电子签 |
| 签字章 | 导师授权使用 |
| 代签 | 导师口头同意，授权代签 |

---

## 沟通话术

> "老师，XX 报销单/申请表需要您签字，金额 XX，走 XX 经费。方便的话我过来找您，不方便我用电子签/代签？"
