# 学校公章（项目申报用印）

> 项目申报需要盖学校公章？先判断涉密还是非密，两条路线。

---

## 🟢 非涉密项目：走线上

```
科研管理服务平台 → 申请人提交 → 科研干事 → 学院领导 → 财务处(如需预算审批) → 科研院 → 用印申请单 → 师生服务大厅盖章
```

---

## 🔴 涉密项目：走线下

```
打印进款审批单 → 填写信息 → 交学院科研干事 → 学院领导 → 财务处审批 → 逸夫楼科研院 → 用印申请单 → 师生服务大厅盖章
```

---

## 关键区别

| | 非密 | 涉密 |
|------|------|------|
| 系统 | 科研管理服务平台（线上） | 不走系统 |
| 表格 | 系统生成 | 手动打印进款审批单 |
| 流转 | 线上逐级审批 | 纸质逐级签字 |
| 最终盖章 | 师生服务大厅 | 师生服务大厅 |
