# 采购合同公章

> 采购需要签合同盖章？先判断金额和项目类型。

---

## 要不要签合同？

根据 BIT 财务报销指南 1.8.1 节：

| 情形 | 是否需合同 |
|------|:---:|
| 政府采购 | ✅ 采购合同 + 电子验收单 |
| 单次交易 **≥ 2 万元** | ✅ 必须提供合同 |
| 分期付款 | ✅ 合同 + 分期付款清单 |
| 涉密科技合同 | ✅ 用审批表代替合同 |
| 外协 / 外拨 | ✅ 必须签合同 |
| 日常办公用品（< 2 万） | ❌ 发票即可 |

### 采购金额分级（科研仪器设备）

| 金额 | 采购方式 | 关键要求 |
|------|----------|----------|
| ≤ 20 万元 | 自行采购 | 直接买 |
| 20-50 万元 | **自行比价** | 3 人比价小组 + 3 家以上供应商报价 + 出具比价报告 |
| ≥ 50 万元 | **招采中心组织** | 校级评比 / 公开招标 |

> ⚠️ ≥ 50 万元：不是你说了算，走招采中心（计划财务部招标采购中心），流程长，提前规划！

### 比价采购流程（20-50 万元科研仪器设备）

1. **建比价小组**：3 人（采购人 + ≥2 名中级及以上职称专家）
2. **获取报价**：选 3 家以上供应商，拿加盖公章的报价单
3. **比价**：选最低价，出具《科研仪器设备比价采购报告》
4. **系统操作**：采购管理系统 → 提交采购申请 → 选「比价」→ 上传比价报告 + 3 家报价单
5. **合同签署**：审批通过后签合同（参考《货物类采购合同示范文本》）
6. **合同盖章**：系统提交合同申请 → 盖章

### 合同盖章要求

双方均需加盖规范印章。我方印章：
- 北京理工大学公章
- 北京理工大学科技合同专用章
- 北京理工大学合同专用章

（详见《北京理工大学合同管理办法》北理工发〔2019〕45 号）

## 合同盖章流程

```
准备合同文本 → 双方确认内容 → 系统提交用印申请 → 学院审核 → 科研院审核 → 用印编号 → 盖章
```

---

## 🟢 非涉密采购合同：走线上

1. 科研管理服务平台 → 提交合同用印申请
2. 上传：合同扫描件 + 采购申请审批单（如需）
3. 学院线上审核 → 科研院线上审核
4. 系统生成用印编号
5. 凭用印编号 → 师生服务大厅盖章

---

## 🔴 涉密采购合同：走线下

1. 打印合同 + 进款/出款审批表
2. 交学院科研干事 → 学院领导审核
3. 逸夫楼科研院复核
4. 科研院生成用印编号
5. 凭用印编号 → 师生服务大厅盖章

---

## 合同盖章后归档

| 项目类型 | 归档要求 |
|----------|----------|
| 横向项目 | 合同原件 1 份 + 备案表 → 逸夫楼 101 董怡 |
| GF / 纵向项目 | 合同原件 1 份 → 项目主管归档（⚠️ 归档后才能办出款！） |
| 一般采购 | 合同 + 发票一起报销 |

---

## 合同原件 vs 复印件

根据 BIT 财务报销指南 1.8.2：

### 一次性付款

| 方式 | 借款时 | 核销/报销时 |
|------|:---:|:---:|
| 先借款再核销 | 复印件 | 原件 |
| 直接报销 | — | 原件 |

### 分期付款

| 方式 | 非最后一次 | 最后一次 |
|------|:---:|:---:|
| 借款再逐笔核销 | 复印件 | 原件 |
| 逐笔借款一次性核销 | 复印件 | 原件 |
| 不借款分期报销 | 复印件 | 原件 |

## 常见错误 ⚠️

| # | 错误 | 预防 |
|---|------|------|
| 1 | 没签合同就买了设备 | 采购前确认是否需要合同 |
| 2 | 合同没归档就办出款 | ⚠️ 归档是出款前置条件 |
| 3 | 合同金额与发票不一致 | 签合同前确认金额 |
| 4 | 涉密合同走了线上 | 涉密一律走线下纸质 |
