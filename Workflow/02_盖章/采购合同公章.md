# 采购合同公章

> 采购需要签合同盖章？先判断金额和项目类型。

---

## 要不要签合同？

根据 BIT 财务报销指南 1.8 节：

| 情形 | 是否需合同 |
|------|:---:|
| 单笔金额较小（日常办公用品等） | ❌ 发票即可 |
| 单笔超过规定金额门槛 | ✅ 必须提供合同 |
| 购买仪器设备 / 固定资产 | ✅ 必须签合同 |
| 外协 / 外拨 | ✅ 必须签合同 |

> 📌 具体金额门槛咨询计财部或科研院，不同经费类型标准不同。

---

## 合同盖章流程

```
准备合同文本 → 双方确认内容 → 系统提交用印申请 → 学院审核 → 科研院审核 → 用印编号 → 盖章
```

---

## 🟢 非涉密采购合同：走线上

1. 科研管理服务平台 → 提交合同用印申请
2. 上传：合同扫描件 + 采购申请审批单（如需）
3. 学院线上审核 → 科研院线上审核
4. 系统生成用印编号
5. 凭用印编号 → 师生服务大厅盖章

---

## 🔴 涉密采购合同：走线下

1. 打印合同 + 进款/出款审批表
2. 交学院科研干事 → 学院领导审核
3. 逸夫楼科研院复核
4. 科研院生成用印编号
5. 凭用印编号 → 师生服务大厅盖章

---

## 合同盖章后归档

| 项目类型 | 归档要求 |
|----------|----------|
| 横向项目 | 合同原件 1 份 + 备案表 → 逸夫楼 101 董怡 |
| GF / 纵向项目 | 合同原件 1 份 → 项目主管归档（⚠️ 归档后才能办出款！） |
| 一般采购 | 合同 + 发票一起报销 |

---

## 合同原件 vs 复印件

根据 BIT 财务报销指南 1.8.2：

| 情形 | 用原件还是复印件 |
|------|:---:|
| 首次报销 | 合同原件 |
| 同一合同多次报销 | 首次原件，后续可用复印件（注明原件存放处） |

---

## 常见错误 ⚠️

| # | 错误 | 预防 |
|---|------|------|
| 1 | 没签合同就买了设备 | 采购前确认是否需要合同 |
| 2 | 合同没归档就办出款 | ⚠️ 归档是出款前置条件 |
| 3 | 合同金额与发票不一致 | 签合同前确认金额 |
| 4 | 涉密合同走了线上 | 涉密一律走线下纸质 |
