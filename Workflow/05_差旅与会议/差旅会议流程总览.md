# 差旅与会议流程总览

> 出差、参会，先看这里。

---

## 常见场景

| 场景 | 需要准备 | 报销类型 |
|------|----------|----------|
| 国内学术会议 | 注册 + 交通 + 住宿 | 差旅报销 |
| 国际会议 | 注册 + 机票 + 住宿 + 签证 | 差旅报销 |
| 调研 / 实验出差 | 交通 + 住宿 | 差旅报销 |
| 线上会议 | 注册费 | 会议注册费报销 |

---

## 出行前清单

- [ ] 出差申请是否已审批？
- [ ] 经费是否已确认？
- [ ] 行程是否已定（机票 / 火车票 / 酒店）？
- [ ] 会议注册是否已完成？
- [ ] 是否需要邀请函 / 会议通知？
- [ ] 签证是否已办？（国际）

---

## 出差后清单

- [ ] 收集所有票据（机票行程单、火车票、住宿发票、会议费发票）
- [ ] 填写出差报销单
- [ ] 附上会议通知 / 邀请函
- [ ] 导师签字
- [ ] 走报销流程 → [[../01_报销/差旅报销|差旅报销]]

## 常见坑

- ❌ 没提前申请出差审批，回来报不了
- ❌ 住宿超标了，只能报标准内部分
- ❌ 发票丢了 → 提前拍照备份！
- ❌ 国际会议忘了换汇证明
