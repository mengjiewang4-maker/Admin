# 报销提交前检查清单

> 每次提交报销前，逐项打勾，防止被退回。
> 基于 BIT 财务报销指南整理。

---

## 一、付款方式检查

- [ ] 是否使用了公务卡或公对公转账？（⚠️ 几乎全部因公支出都应走公务卡）
- [ ] 公务卡刷卡记录是否已认证报销？（还款期限：刷卡次月 25 日）
- [ ] 如果是公对公汇款：是否拿到了对方账户信息？

---

## 二、发票检查

- [ ] 发票是否齐全？少一张都不行
- [ ] 发票抬头：**北京理工大学**（全称）
- [ ] 税号：**12100000400009127B**
- [ ] 发票金额与报销单金额是否一致？
- [ ] 电子票据 → 是否已打印？（黑白/彩色均可）
- [ ] 纸质票据 → 原件（复印件、照片打印件无效！）
- [ ] 电子票据 or 单张≥1000 元纸质发票 → 是否已在系统**查验认证**？
- [ ] 发票背面是否已签名？

---

## 三、系统填报检查

- [ ] 是否登录「高级财务管理平台」→「网上预约报账」→「申请报销单」？
- [ ] 是否完成「增值税发票查验」？
- [ ] 预约费用项是否选对？
- [ ] 经费卡号 / 项目号（13 位代码）是否正确？
- [ ] 预算科目是否正确？
- [ ] 是否打印了系统生成的「北京理工大学网上报账单」？

---

## 四、材料检查

- [ ] 是否附上相关审批单？
- [ ] 是否附上会议通知 / 邀请函？（会议费）
- [ ] 是否附上行程单 / 登机牌？（差旅费）
- [ ] 是否附上采购申请单？（采购类）
- [ ] 是否附上合同？（单笔超过规定金额）
- [ ] 是否附上备案表 + 完整签字？（专利费）⚠️
- [ ] 专利老师发来的材料是否一份不落全交了？（专利费）⚠️
- [ ] 所有材料是否按顺序整理好？

---

## 五、签字检查

- [ ] 经费负责人（导师）是否已签字？
- [ ] 经手人是否已签字？
- [ ] 验收人是否已签字？（采购类）
- [ ] 科研经费支出 → 经费负责人签字
- [ ] 非科研经费支出 → 单位负责人签字

---

## 六、票据丢失处理（如适用）

- [ ] 是否已向开票单位索取记账联复印件 + 加盖发票专用章？
- [ ] 是否填写「报销票据丢失情况说明」？
- [ ] 是否已请核算室负责人审批？
- [ ] 火车票/飞机票丢失：是否有网上订单及付款截图？

---

## 七、最后检查

- [ ] 所有材料是否已保存电子版 / 扫描件？
- [ ] 是否有之前被退回的同类问题？→ 查 [[../09_问题与复盘/问题库|问题库]]
- [ ] 是否知道最近的自助报账投递机位置？（中关村/良乡）
- [ ] 差旅：补贴和打车费是否二选一？（不能同时报）⚠️
- [ ] 差旅：火车票金额是否按 12306 实际支付算？（非票面金额）⚠️
- [ ] 专利费：备案表是否完整签字？专利老师发的材料全交了吗？⚠️

---

## 八、投递前确认

- 📁 打印好的「北京理工大学网上报账单」
- 📁 所有原始票据 + 附件材料
- 📱 手机里留电子版备份

> ⚠️ 如果不确定某一步，先看 [[报销流程总览|报销流程]]，或查 [[../09_问题与复盘/问题库|问题库]]。
