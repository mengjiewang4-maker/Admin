# 横向开发票与经费上账

> 适用范围：增值税普通发票（电子），增值税专用发票（电子）
> 更新日期：2025 年 9 月
> 
> ⚠️ 横向项目开的是**增值税发票**，不是收据！

---

## 1. 适用场景

- 横向科研项目需要给合作单位开发票
- 开票后完成经费上账（认领款项）

---

## 2. 两条路线速查

```
款已到 → 开票 + 上账（发票开出 3 个工作日后才能上账）
款未到 → 先开预借发票 → 等款到 → 核销借票 + 上账
```

---

## 3. 🟢 路线一：款项已到（开到款发票 + 上账）

### 第一步：线上申请开票

登录**财务综合信息门户** → 「预开票系统」→ 线上申请开票

> 📌 2025 年 4 月计财部系统升级：新增**录入合同信息**步骤！

提交后进入审批流程，项目负责人审批。

操作指南：`《预开票系统操作指南 3.5-250325》`

### 第二步：上账（发票开出 3 个工作日后）

提交上账材料给学院科研财务助理：

| # | 材料 | 备注 |
|---|------|------|
| 1 | 银行进账单 | 财务系统打印 |
| 2 | 到款凭证 | 仅 24 年到款需要 |

线上认领款项（横向非 M）：
- 科研管理服务平台 → 认领款项

开票和上账同步完成 ✅

---

## 4. 🟡 路线二：款项未到（预借发票 → 核销上账）

### 第一阶段：开预借发票

财务综合信息门户 → 「预开票系统」→ 线上申请

系统升级新操作：录入合同信息。

### 第二阶段：经费到校后核销 + 上账

1. 提交上账材料：银行进账单（+ 24年到款凭证）
2. ⚠️ 在财务系统提交**核销借票申请**（必须在经费未到个人账户前提交！）
3. 科研管理服务平台 → 认领款项

---

## 5. ⚠️ 税务计算（横向项目必须了解）

### 税款公式

```
增值税税额 = 票面金额 / (1 + 税率) × 税率
附加税费   = 增值税税额 × 12%
应缴税额   = 增值税税额 + 附加税费
```

### 常用税率

| 合同类型 | 税率 | 备注 |
|----------|:---:|------|
| 销售货物、劳务（加工/修理） | **13%** | |
| 销售服务 | **6%** | |
| 软件（开发/服务） | **6%** | |
| 技术（开发/咨询/转让/服务） | **3%** | 简易征收，不能增值税进项抵扣 |

### 票面限额

| 发票类型 | 单张限额 |
|----------|:---:|
| 增值税普通发票（含电子/免税） | < 100 万元 |
| 增值税专用发票 | < 1000 万元 |

> 超出限额需拆分成多张发票。

---

## 6. ⚠️ 时间节点

| 规则 | 说明 |
|------|------|
| 每月 25 日后 | 停止开票 |
| 发票开出后 3 个工作日 | 才能办理上账 |
| 每周四 | 科研财务老师前往计财部 |
| 核销借票申请 | ⚠️ 必须在经费未到个人账户前提交 |

---

## 7. 合同认定业务

提交申请至科技项目管理中心（逸夫楼）：

**线上（公开合同）**：科研管理服务平台 → 上传：
1. 合同原件扫描件（双方签字盖章，在有效期内）
2. 项目预算页

---

## 8. 合同免税业务

> ⚠️ **专票无法开免税！**

### 办理流程

1. 科研管理服务平台 → 在线填写免税信息 → 上传：
   - 银行进账单 / 预开免税发票承诺书
   - 双方盖章合同及附件
   - 项目技术方案 1 份（非首次仅需①）

2. 科技项目管理中心审核通过 → 线上下载

### 财务系统开免税发票

上传 4 个文件（存为 `合同名称-XX学院` 文件夹）：
1. 《技术合同登记证明》（一个合同一次）
2. 《免税发票信息备案表》（每次更新）
3. 《技术合同技术性收入核定表》（每次更新）
4. 合同扫描件

## 9. 常见问题 Q&A

### Q1：如何选择票据类型？

| 项目类型 | 开票类型 |
|----------|----------|
| 纵向科研 | 中央行政事业单位资金往来结算票据 / 银行到款凭证复印件加盖财务章 |
| 横向科研 | 增值税普通发票（含电子）或专用发票 |
| 横向免税 | 增值税普通发票（含电子） |
| 其他教育服务 | 根据合同内容确定 |

### Q2：横向经费卡不够缴税 / 没有横向经费卡？

- 方案 1：申请垫付 → 填写《垫付税款申请书》→ 求是楼 418 审核
- 方案 2：借用他人经费卡 → 开票申请表附所借卡负责人电子签

### Q3：甲方不需要收据？

填写《不开票据承诺书》，项目负责人签字 → 求是楼 418 审核。

### Q4：需要退换票据？

查阅《关于退（换）票据流程的说明》，填写《北京理工大学退（换）票据申请表》，项目负责人签字 → 求是楼 418 审核。

### Q5：合同 SM 无法上传？

上传合同说明模板（m），加盖学院公章，上传到系统。

### Q6：发票内容为货物销售，系统没合适选项？

在《商品和服务税收分类编码》中选择货物编码（选到最后一位），填写《货物名称》，发送给科研财务老师**朱文静**。

### Q7：如何计算税额？

增值税 = 票面金额 / (1+税率) × 税率，附加税 = 增值税 × 12%。

## 11. 常见错误 ⚠️

| # | 错误 | 预防 |
|---|------|------|
| 1 | 横向项目开了收据而不是发票 | 横向 = 增值税发票，纵向 = 收据 |
| 2 | 发票没满 3 个工作日就去上账 | ⚠️ 开票后等 3 天再上账 |
| 3 | 核销借票申请提交晚了 | 核销必须在经费未到个人账户前提交 |
| 4 | 25 号以后去开发票 | 每月 25 日后停止开票 |
| 5 | 专票想开免税 | ❌ 专票无法免税 |
| 6 | 票面金额超限额没拆分 | 普票 < 100 万，专票 < 1000 万 |
| 7 | 退换票影响入账进度 | 开票前务必确认对方信息无误 |
