## 报销前必做

1. 确认经费余额 → 问导师或查系统
2. 确认预算科目允许 → 别走到不允许的科目
3. 看 [[清单_报销提交前|报销提交前检查清单]]

## 通用流程速览（BIT 实际）

```
收集票据 → 登录「高级财务管理平台」预约网上报销单 → 打印报账单
    → 完成审签（导师/经费负责人签字）→ 投递至自助报账投递机 → 等打款
```

系统：**高级财务管理平台**（"网上预约报账" → "申请报销单"）
视频教程：计划财务部 → 会计核算中心 → 服务指南 → "计小财的报账路"

## ⚠️ 关键规则（来自 BIT 财务报销指南）

### 付款方式

| 方式 | 说明 |
|------|------|
| **公务卡** | ⭐ 推荐！几乎所有因公支出都用公务卡 |
| 对公汇款 | 学校账户直接打款至对方单位 |
| 对私支付 | 打款至个人绑定银行卡 |

> 强烈建议：**不区分经费、不区分费用类型、不区分金额，所有因公支出都使用公务卡或公对公转账。**

### 必须使用公务卡的费用

办公费、图书资料费、印刷费、咨询费、手续费、水电费、邮电费、差旅费、维修费、租赁费、会议费、培训费、公务接待费、设备费、公车运行维护费，等。

### 发票查验

- 电子票据 → 必须查验认证
- 单张发票金额 ≥ 1000 元的纸质通用发票 → 必须查验认证
- 已丢失仍申请报销的纸质票据（火车票/飞机票除外）→ 必须查验认证

**查验方式**：高级财务管理平台 → "增值税发票查验"功能

### 票据丢失

1. 向开票单位索取记账联复印件 + 加盖对方发票专用章
2. 填写「报销票据丢失情况说明」
3. 计划财务部会计核算中心核算室负责人审批
4. 火车票/飞机票丢失：提供网上订单及付款截图，或票务中心证明
