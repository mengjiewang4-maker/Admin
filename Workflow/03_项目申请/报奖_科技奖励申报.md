# 科技奖励申报（报奖）盖章及公示流程

> 北京理工大学科研院 · 科技奖励申报全流程
> 系统入口：[科研管理服务平台](http://kyxt.bit.edu.cn/)
> 
> ⚠️ **先判断：你的项目是「涉密」还是「非密」？两条路线不同！**

---

## 1. 适用场景

申报各类科技奖励时需要走校内流程。

适用范围：
- 国家科学技术奖
- 省部级科学技术奖（如北京市科学技术奖）
- 学会/协会科技奖（如中国颗粒学会自然科学奖）
- 其他需学校盖章的科技奖励申报

---

## 2. 流程速查：非密项目 vs 涉密项目

### 🟢 非密项目流程（大多数情况）

```
①保密审查审批表 → ②公示情况说明 → ③登录kyxt系统填报 → ④学院+科研院线上审核 → ⑤系统分配用印编号 → ⑥教师服务大厅盖章
```

### 🔴 涉密项目流程（涉M项目专用）

```
①合作报奖情况说明 → ②科研院长签字+学院盖章 → ③内容脱密 → ④公示情况说明 → ⑤保密审查审批表 → ⑥携带纸质版至科研院办理用印和公示
```

## 4. 🟢 非密项目 · 详细步骤

### 步骤 1：保密审查

填写「公开信息保密审查审批表」→ 学院审核 → 保密办公室审核通过。

| 环节 | 谁做 | 怎么做 |
|------|------|--------|
| 申请人自查 | 你 | 签字承诺内容不涉密 |
| 单位定密责任人审查 | 学院定密责任人 | 给出"同意公开"或"不同意公开"意见 |
| 保密办备案 | 保密办公室 | 签字/盖章 |

📌 **一式两份**，申请人 + 保密办各留一份。

---

### 步骤 2：公示（如项目需要公示）

填写「报奖公示情况说明」→ **项目负责人签字**。

公示通知起草 → 提交科研院在校内发布。

---

### 步骤 3：系统填报（关键！）

登录 [科研管理服务平台](http://kyxt.bit.edu.cn/)：

```
进入「科研成果」模块 → 新增「科技奖励」→ 按要求填写奖励项目基本信息 → 上传需盖章材料
```

---

### 步骤 4：线上审核

**学院** → **科研院** 线上逐级审核。

---

### 步骤 5：系统分配用印编号

审核通过后，系统自动分配**「用印申请编号」**。

---

### 步骤 6：去教师服务大厅盖章

携带需盖章的纸质材料 + 用印申请编号 → **教师服务大厅**办理用印。

> 📍 注意：非密项目是去**教师服务大厅**盖章，不是去逸夫楼！

---

## 5. 🔴 涉密项目 · 详细步骤

### 步骤 1：合作报奖情况说明

填写「北京理工大学合作报奖情况说明」→ **我校牵头完成人签字确认**信息及排序无误。

---

### 步骤 2：学院审批盖章

我校牵头完成人所在学院 → **科研院长审核签字** → **加盖学院公章**。

---

### 步骤 3：公示（如需）

将申报材料**内容脱密** → 填写「报奖公示情况说明」→ **项目负责人签字**。

---

### 步骤 4：保密审查

填写「公开信息保密审查审批表」→ **学院**审核 → **保密办公室**审核通过。

---

### 步骤 5：科研院办理用印和公示

携带所有**纸质版材料**至**科研院**办理用印和公示手续。

> 📍 注意：涉密项目是去**科研院**，不走线上系统，不经过教师服务大厅。

## 8. 时间规划建议

| 时间节点 | 要做什么 |
|----------|----------|
| 奖励通知发布后立即 | 判断涉密/非密，确认申报资格 |
| 提交截止前 3-4 周 | 完成申报书初稿 |
| 提交截止前 3 周 | 启动保密审查 |
| 提交截止前 2 周 | 提交公示材料 → 发布公示 |
| 公示期内（3-7 天） | 处理可能出现的异议 |
| 公示结束后 | 完成公示情况说明签字 |
| 提交截止前 1-2 周 | **非密**：系统填报 + 线上审核 |
| | **涉密**：纸质版提交科研院 |
| 提交截止前 1 周 | 完成盖章（非密去教师服务大厅，涉密去科研院） |
| 提交截止前 3 天 | 提交电子版（如有） |
| 提交截止日 | 纸质材料寄出/送达 |

> ⚠️ **公示需要时间，不能卡到最后一周才开始校内流程！**

## 9. 常见错误 ⚠️

| # | 错误 | 原因 | 预防 |
|---|------|------|------|
| 1 | 非密项目跑去了科研院，涉密项目跑去了教师服务大厅 | 没分清两条路线 | **先判断涉密/非密！** |
| 2 | 忘了先做保密审查就公示 | 不知道保密审查是前置步骤 | **绝对第一步！** |
| 3 | 没在 kyxt 系统填报就直接去盖章（非密） | 不知道系统流程 | 非密项目必须先线上审核、拿到用印编号 |
| 4 | 涉密项目没做脱密处理就公示 | 不知道脱密要求 | 涉密项目公示前必须内容脱密 |
| 5 | 公示期不够 | 没仔细看奖励通知 | 先确认公示天数要求 |
| 6 | 公示内容与申报书不一致 | 改了内容没更新公示 | 公示内容 = 最终申报内容 |
| 7 | 保密审查审批表没留底 | 只给保密办存了 | **自留一份！** |

## 11. 沟通话术 💬

### 非密项目 · 咨询系统填报

> "老师您好，我们在申报 XX 奖，是**非密项目**。保密审查已经通过了，请问现在在 kyxt 系统里具体怎么操作？是进「科研成果」→「新增科技奖励」对吗？"

### 涉密项目 · 提交纸质版

> "老师您好，我们是 XX 项目的，是**涉密项目**，需要申报 XX 奖。合作报奖情况说明和保密审查都已经完成了，纸质材料带过来了，麻烦您帮我们办理用印和公示。"

### 去教师服务大厅盖章（非密）

> "老师您好，我来办理科技奖励申报的用印。系统用印申请编号是 XXXX，材料都在这里了。"
