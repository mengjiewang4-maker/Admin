# 科工局非涉密项目 · 全流程

> 来源：北京理工大学 GF 纵向项目业务流程
> 对应上级主管部门：国家国防科技工业局（科工局）
> 项目属性：非涉密（通用类）
> 
> ⚠️ 科工局项目走「专用项目管理系统」，不走 kyxt。

---

## 1. 适用场景

- 科工局主管的国防纵向科研项目（非涉密 / 通用类）
- 包括：技术基础、基础科研、军品配套等
- 经费文件依据：12 号文 / 18 号文 / 8 号文

---

## 2. 流程总览

```
①申报 → ②正式立项 → ③过程管理(里程碑/年度考核等) → ④出款(外协/外拨) → ⑤结题验收
```

每一步都走：**专用系统填报 → 学院审核 → 计财部审核（经费相关）→ 科研院审核 → 系统生成用印编号 → 盖章**

---

## 3. 所需材料速查

| 阶段 | 核心材料 | 模板/系统 |
|------|----------|-----------|
| 申报 | 申报书、经费预算表、进款审批表 | 专用系统生成 |
| 立项 | 合同/任务书/协议书、立项审批表 | 专用系统生成 |
| 过程管理 | 里程碑报告、年度报告、测试大纲等 | 按上级要求 |
| 出款 | 出款合同、乙方资质、出款审批表 | 专用系统生成 |
| 结题 | 验收材料（技术+财务） | 按验收要求 |

## 4. 阶段一：申报阶段

### 流程图

```
项目负责人在专用系统填报 → 学院审核(定密+材料) → 计财部审核(预算) → 科研院审核 → 系统生成用印编号 → 教师服务大厅盖章
```

### 4.1 系统填报

登录**专用项目管理系统**：

```
新增项目 → 勾选「申报阶段」+「GF纵向」
```

填写：
- 申报项目基本信息
- 用印事项及材料明细
- 用印类型及数量
- 选择财务预算模板（12号文 / 18号文 / 8号文）

提交后系统生成**业务追踪号**。

### 4.2 打印进款审批表

系统打印进款审批表 → 在纸质表格中**补充填写**：
- 项目名称
- 指南编号
- 项目密级（非密项目填"非密"或"公开"）

### 4.3 学院审核

- 携带：进款审批表 + 申报材料
- 线下至学院完成：定密确认 + 申报材料审核
- 学院根据业务追踪号完成线上审批，编制申报号
- 审核流转至计财部

### 4.4 计财部审核（预算）

- 计财部根据系统预算表线上审核经费预算
- 申报人从系统打印**计财部用印申请单**，办理财务用印
- 计财部完成审核，流转至科研院

### 4.5 科研院审核

- 携带：进款审批表 + 申报相关材料
- 线下至科研院进行复核
- 科研院完成线上审批，系统生成**用印编号** ✅

### 4.6 办理盖章

凭**用印编号** + 申报相关材料 → **教师服务大厅**办理盖章。

申报程序结束。

---

### 🔁 申报阶段再次用印

如果申报阶段需要再次盖章：

```
专用系统 → 申报阶段再次用印端口 → 填写再次用印事项 → 学院审核 → 科研院审核 → 用印编号 → 教师服务大厅盖章
```

## 5. 阶段二：正式立项阶段

### 流程图

```
项目负责人在专用系统填报 → 学院审核(定密+材料+合同) → 计财部(经费预算) → 科研院审核(合同用印) → 用印编号 → 盖章
```

### 5.1 系统填报

项目获得上级主管部门批复后，登录**专用项目管理系统**：

```
新增项目 → 勾选「正式立项」+「GF纵向」
```

选择：
- 勾选「生产」（如涉及生产任务）
- 勾选「主项」或「子项」
- 填写档案号（如已有）

填写：
- 项目基本信息（名称、指南编号、密级等）
- 经费预算
- 合同 / 任务书 / 协议书等用印事项及材料明细
- 用印类型及数量

提交后生成业务追踪号。打印立项审批表。

### 5.2 学院审核

- 携带：立项审批表 + 立项材料 + 合同/任务书/协议书 + 密级相关材料
- 线下至学院完成：**定密** + 材料审核 + **合同内容审核**
- 学院根据业务追踪号完成线上审批，编制档案号
- 审核流转至计财部

### 5.3 计财部审核

- 计财部根据系统预算表线上完成经费预算审核
- 申报人从系统打印**计财部用印申请单**，办理财务用印
- 审核流转至科研院

### 5.4 科研院审核 + 盖章

- 携带：立项审批表 + 合同/任务书/协议书 + 密级相关材料
- 线下至科研院复核
- 科研院完成线上审批，系统生成**用印编号**
- 凭用印编号 + 合同/任务书/协议书等材料 → 根据用印类型至**科研院**或**教师服务大厅**办理盖章

> 📌 合同需甲乙双方盖章。合同归档：提交 1 份至项目主管归档。

## 6. 阶段三：过程管理审核及用印

### 适用场景

- 里程碑节点考核
- 年度考核
- 测试大纲评审
- 其他上级主管部门或委托方要求的审查节点

### 流程

```
项目负责人准备材料 → 专用系统填报用印申请 → 学院审核 → [财务审查(如需)] → 科研院审核 → 用印编号 → 盖章
```

### 6.1 系统填报

```
专用系统 → 正式立项过程管理端口 → 填写具体用印事项
```

- 填写具体事项及材料明细、用印类型及数量
- 提交后生成业务追踪号

### 6.2 学院审核

- 项目负责人携带具体事项相关材料，线下至学院审核
- 学院根据上级主管部门和委托方要求，完成线上审批

### 6.3 财务审查（条件触发）

⚠️ 如果过程管理材料包含年度/节点的**财务支出报告/报表**：

- 项目负责人携带材料至计财部线下审核
- 计财部加盖财务公章
- 审核通过后进入下一步

### 6.4 科研院审核 + 盖章

- 项目负责人携带过程管理材料 → 线下至科研院复核
- 科研院完成线上审批，系统生成用印编号
- 凭用印编号 → 根据用印类型至**科研院**或**教师服务大厅**盖章

---

## 7. 阶段四：出款审核及用印

### 前置条件

> ⚠️ 主项目的正式合同/任务书/协议书**必须先完成归档**，才能办理出款！

### 适用场景

- 项目外协（委托外单位协作）
- 项目外拨（经费拨付合作单位）

### 流程

```
项目负责人准备出款材料 → 专用系统出款审批 → 学院审核(定密+资质) → 科研院审核 → 用印编号 → 盖章
```

### 7.1 准备材料

| # | 材料 | 说明 |
|---|------|------|
| 1 | 出款合同文本 | 与外协/外拨单位签订的合同 |
| 2 | 乙方单位资质材料 | 营业执照、保密资格证书（涉密外协时）、保密监督检查表（涉密外协时） |
| 3 | 保密协议书 | ⚠️ 仅涉密外协/外拨时需要 |

> 📌 非涉密项目外协/外拨，**不需要**保密资格证书和保密协议书。

### 7.2 系统填报

```
专用系统 → 出款新增端口 → 先完成出款承诺 → 依托主项目 → 填写出款合同基本信息
```

提交后生成业务追踪号。打印**出款审批表**，在纸质表中补充：
- 主项目名称
- 密级（非密项目填"非密"）
- 定密依据

### 7.3 学院审核

- 携带：出款审批表 + 出款合同 + 乙方资质材料
- 线下至学院完成定密和材料审核
- 学院根据业务追踪号完成线上审批

### 7.4 科研院审核 + 盖章

- 携带：出款审批表 + 出款合同 + 乙方资质材料（非密项目无需保密协议书）
- 线下至科研院复核
- 科研院完成线上审批，系统生成用印编号
- 凭用印编号 → 根据用印类型至**科研院**或**教师服务大厅**盖章

---

## 8. 阶段五：结题审核及用印

### 前置条件

项目已按照合同/任务书/协议书约定**完成所有内容**。

### 流程

```
项目负责人准备验收材料 → 专用系统填报结题用印 → 学院审核 → [财务验收(如需)] → 科研院审核 → 用印编号 → 盖章 → 归档
```

### 8.1 准备验收材料

按照上级主管部门或委托方关于技术、财务验收的要求准备。

### 8.2 系统填报

```
专用系统 → 结题验收申请界面 → 填写结题用印事项及材料明细
```

提交后生成业务追踪号。

### 8.3 学院审核

- 携带验收相关材料 → 线下至学院审核
- 学院完成线上审批

### 8.4 财务验收审核（条件触发）

⚠️ 如果验收材料包含：
- 财务验收报告 / 报表
- 财务审计报告
- 财务整改报告

则需：
- 项目负责人携带材料至**计财部**线下审核
- 计财部加盖财务公章

### 8.5 科研院审核 + 盖章

- 携带验收材料 → 线下至科研院复核
- 科研院完成线上审批，系统生成用印编号
- 凭用印编号 → 根据用印类型至**科研院**或**教师服务大厅**盖章

### 8.6 归档 ⚠️ 别忘了！

项目结题后，项目团队应及时梳理**项目合同 + 验收相关材料**，按照学校档案馆归档要求及时归档。

## 9. 审批链速查（非密项目，所有阶段通用）

```
系统填报 → 生成业务追踪号
    ↓
学院审核（线下 + 线上）
    ↓
计财部审核（涉及预算/经费时）  ← 不是每步都走
    ↓
科研院审核（线下 + 线上）
    ↓
系统生成用印编号
    ↓
教师服务大厅 / 科研院盖章
```

## 12. 非密 vs 涉密关键区别

| 事项 | 非密项目（通用类） | 涉密项目（专用类） |
|------|-------------------|---------------------|
| 定密 | 确认非密即可 | 逐级定密审查 |
| 保密协议书（出款） | ❌ 不需要 | ✅ 必须提供 |
| 乙方保密资质（出款） | ❌ 不需要 | ✅ 必须提供 |
| 系统 | 专用项目管理系统（同） | 专用项目管理系统（同） |
| 盖章地点 | 教师服务大厅 or 科研院 | 以科研院为主 |

## 13. 常见错误 ⚠️

| # | 错误 | 原因 | 预防 |
|---|------|------|------|
| 1 | 项目归档前就开始办出款 | 不知道出款需要主项目先归档 | **先确认主合同已归档** |
| 2 | 非密出款多带了保密协议和保密资质 | 按涉密流程办了 | 非密出款不需要保密材料 |
| 3 | 跳过学院审核直接找科研院 | 不知道学院必须先批 | 学院 → 计财部 → 科研院，顺序不能乱 |
| 4 | 忘记系统生成业务追踪号就去线下办 | 以为线下直接交就行 | **先在系统提交！** |
| 5 | 盖章时没带用印编号 | 忘了系统生成的编号 | 截图保存用印编号 |
| 6 | 结题后忘了归档 | 以为交了验收材料就完事 | 结题 = 验收 + 归档 |
| 7 | 预算选错模板（12号文 vs 18号文 vs 8号文） | 不清楚经费文件依据 | 申报前先确认经费文件 |
