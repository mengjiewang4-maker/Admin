# 项目申请提交前检查清单

> 提交项目申请前最后一遍检查。截止时间不等人。

---

## 一、材料完整性

- [ ] 申请书是否完整填写？
- [ ] 预算表是否填写？
- [ ] 团队成员信息是否完整？
- [ ] 合作单位材料是否齐全？（如有）
- [ ] 伦理审查材料？（如需要）
- [ ] 推荐信 / 导师意见？（如需要）
- [ ] 附件材料是否齐全？

---

## 二、内容检查

- [ ] 项目名称是否与指南要求一致？
- [ ] 申请人信息是否准确？
- [ ] 经费预算是否合理？（别超单项限额）
- [ ] 正文是否在字数 / 页数限制内？
- [ ] 格式是否符合要求（字体、字号、行距）？
- [ ] 参考文献格式是否正确？

---

## 三、签字盖章

- [ ] 导师是否已签字？
- [ ] 学院是否已盖章？
- [ ] 学校是否已盖章？（如需要）
- [ ] 合作单位是否已盖章？（如有）
- [ ] 所有签字盖章位置是否正确？

---

## 四、系统提交

- [ ] 系统账号密码是否确认？
- [ ] 所有字段是否填写完整？
- [ ] 上传附件是否成功？
- [ ] 预览 / 生成 PDF 是否正常？
- [ ] PDF 是否有缺页、乱码？
- [ ] 提交按钮是否已点击？
- [ ] **是否收到提交成功确认？**（截图保存！）

---

## 五、纸质版（如需要）

- [ ] 纸质版份数是否正确？
- [ ] 纸质版与电子版是否一致？
- [ ] 是否需要送达指定地点？
- [ ] 邮寄是否需要挂号 / 快递单号？

---

## 六、截止时间

- [ ] 校内截止时间：\_\_\_\_ （通常比正式截止早几天！）
- [ ] 系统截止时间：\_\_\_\_
- [ ] 纸质材料截止时间：\_\_\_\_
- [ ] 现在是几点？还剩多少时间？

---

## 七、提交后确认

- [ ] 是否保存了提交截图？
- [ ] 是否通知了导师"已提交"？
- [ ] 是否通知了科研秘书"已提交"？
- [ ] 是否需要跟踪后续审批状态？

---

> 🔗 具体流程看对应项目的申请页面
> ⚠️ 截止日期前几天系统可能卡顿，不要卡到最后一天！
