# 文件命名规则

> 核心原则：**一眼看懂是什么 + 文件列表自动排序**

---

## 一、文件夹编号规则

所有一级文件夹使用 `XX_名称` 格式，数字保证排序：

| 编号 | 文件夹 | 用途 |
|------|--------|------|
| 00 | 总入口 | 日常快速查看 |
| 01 | 报销 | 所有报销相关流程 |
| 02 | 盖章 | 学院 / 学校盖章流程 |
| 03 | 项目申请 | 国自然 / 省基金 / 校内项目 |
| 04 | 合同与采购 | 采购申请、合同签订 |
| 05 | 差旅与会议 | 出差、会议注册、行程安排 |
| 06 | 联系人 | 导师、财务、科研秘书等 |
| 07 | 模板库 | 可复用的笔记模板 |
| 08 | 事务记录 | 每次实际办理的记录 |
| 09 | 问题与复盘 | 踩坑记录、复盘、经验 |
| 10 | 系统 | 命名规则、标签体系等元信息 |

---

## 二、流程类文件命名

格式：`{流程名称}.md`

- 不要加前缀，文件夹本身就是分类
- 流程总览文件统一叫 `XX流程总览.md`
- 具体流程直接描述，用自然语言

**示例：**

```
01_报销/
├── 报销流程总览.md
├── 差旅报销.md
├── 会议注册费报销.md
├── 材料采购报销.md
├── 版面费报销.md
└── 日常交通报销.md
```

---

## 三、事务记录命名

格式：`记录_YYYYMMDD_{事项简称}.md`

- YYYYMMDD = 办理日期
- 事项简称 = 自己能看懂的关键词

**示例：**

```
记录_20260616_差旅报销_北京会议.md
记录_20260620_项目申请盖章_国自然面上.md
记录_20260701_采购_光谱仪.md
记录_20260715_会议注册_APS_March.md
```

**排序效果**：自然按时间排序，找最新/最旧都方便。

---

## 四、模板命名

格式：`模板_{用途}.md`

**示例：**

```
模板_报销流程.md
模板_盖章流程.md
模板_项目申请.md
模板_联系人卡片.md
模板_事务记录.md
```

---

## 五、检查清单命名

格式：`清单_{用途}.md`

**示例：**

```
清单_报销提交前.md
清单_盖章提交前.md
清单_项目申请提交前.md
清单_被退回复盘.md
清单_文件归档.md
```

---

## 六、联系人文件命名

格式：`联系人_{姓名或岗位}.md`

**示例：**

```
联系人_张老师_财务处.md
联系人_李老师_科研秘书.md
联系人_王老师_学院办公室.md
```

---

## 七、快速判断：我该用什么格式？

| 场景 | 命名 |
|------|------|
| 写一个流程 SOP | 直接放对应文件夹，`{名称}.md` |
| 记录一次办理经历 | `记录_YYYYMMDD_{简称}.md` 放 `08_事务记录/` |
| 创建可复用模板 | `模板_{用途}.md` 放 `07_模板库/` |
| 添加检查清单 | `清单_{用途}.md` 放对应流程文件夹 |
| 记录联系人 | `联系人_{姓名}.md` 放 `06_联系人/` |
| 记录踩坑 | 在 `09_问题与复盘/问题库.md` 追加 |
