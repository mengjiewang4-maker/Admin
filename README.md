# Admin

行政事务流程仓库。每一次麻烦事，都要变成下一次的捷径。

## 结构

```
Workflow/
├── 00_总入口.md          ← 日常从这里开始
├── 01_报销/              ← 差旅 / 会议 / 采购 / 版面 / 交通
├── 02_盖章/              ← 学院公章 / 导师签字
├── 03_项目申请/          ← 报奖 / 科工局GF纵向
├── 04_合同与采购/
├── 05_差旅与会议/        ← 差旅会议 / 涉密因私出境
├── 06_联系人/            ← 导师、财务、科研秘书
├── 07_模板库/            ← 报销 / 盖章 / 项目 / 联系人 / 事务记录
├── 08_事务记录/          ← 每次办事的实际经历
├── 09_问题与复盘/        ← 问题库 / 被退回复盘 / 归档
└── 10_系统/              ← 命名规则 / 标签 / 沉淀方法 / AI维护规则
raw/                       ← 原始附件与模板文件
```

## 使用方式

这是一个 Obsidian vault 的子目录，所有 `.md` 文件使用 Wiki-link 互联。

配合 AI（Claude / ChatGPT）维护效果最佳，规则见 `Workflow/10_系统/AI维护规则.md`。
