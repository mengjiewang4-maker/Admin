# 合同与采购流程总览

> 买东西、签合同，先看这里。

---

## 常见采购场景

| 场景 | 需要什么 | 复杂性 |
|------|----------|:---:|
| 小额材料（< 几千） | 发票 + 报销 | ⭐ |
| 中型设备 | 采购申请 + 合同 + 验收 | ⭐⭐⭐ |
| 大型设备 | 招标 / 比价 + 合同 + 验收 | ⭐⭐⭐⭐⭐ |

---

## 采购前必问

- [ ] 经费账号是否允许买这个？
- [ ] 预算够不够？
- [ ] 是否需要走采购申请？
- [ ] 是否需要比价 / 招标？
- [ ] 是否有指定供应商？

---

## 通用流程

```
确认经费 → 提交采购申请 → 审批 → 签合同(如需) → 收货 → 验收 → 报销
```
