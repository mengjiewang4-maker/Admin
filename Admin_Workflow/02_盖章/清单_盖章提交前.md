# 盖章前检查清单

> 出发去盖章前，逐项打勾。白跑一趟是最亏的。

---

## 一、材料检查

- [ ] 所有需要盖章的材料是否已准备齐全？
- [ ] 每份材料需要的份数是否正确？
- [ ] 是否多带 1-2 份备用？（强烈建议）
- [ ] 文件是否已按要求排版 / 打印？
- [ ] 盖章位置是否已标明 / 预留？

---

## 二、签字检查

- [ ] 导师是否已在需要的地方签字？
- [ ] 导师签字是否在正确位置？
- [ ] 是否需要学院审批人先签字？
- [ ] 材料上标注的签字人是否与实际一致？

---

## 三、审批检查

- [ ] 是否需要先在系统里提交申请？
- [ ] 系统审批是否已通过？
- [ ] 是否需要带系统审批通过截图 / 单号？

---

## 四、办理信息确认

- [ ] 办理地点是否确认？（办公室号、窗口号）
- [ ] 办理时间是否正确？（别赶到中午休息）
- [ ] 今天是否工作日？
- [ ] 负责盖章的老师今天是否在？（建议先确认）
- [ ] 是否知道找哪位老师？

---

## 五、备用方案

- [ ] 如果盖章老师不在，有备用时间吗？
- [ ] 如果被要求补充材料，知道找谁吗？

---

## 六、带上这些

- 📁 所有需要盖章的材料 + 备份份
- 🖊 笔
- 🪪 校园卡 / 身份证
- 📱 手机里有电子版
- 📋 系统审批截图（如需要）
