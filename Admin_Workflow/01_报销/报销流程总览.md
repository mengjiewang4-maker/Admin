## 报销前必做

1. 确认经费余额 → 问导师或查系统
2. 确认预算科目允许 → 别走到不允许的科目
3. 看 [[清单_报销提交前|报销提交前检查清单]]

---

## 通用流程速览

```
收集发票 → 填报销单 → 导师签字 → 学院审核(如需) → 提交财务 → 等打款
```

每一步的具体操作，看对应的报销类型流程页。
