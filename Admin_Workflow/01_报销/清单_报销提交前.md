# 报销提交前检查清单

> 每次提交报销前，逐项打勾，防止被退回。

---

## 一、发票检查

- [ ] 发票是否齐全？少一张都不行
- [ ] 发票抬头是否正确？（学校全称）
- [ ] 税号是否正确？
- [ ] 发票金额与报销单金额是否一致？
- [ ] 发票日期是否在规定期限内？
- [ ] 电子发票是否已打印？
- [ ] 发票背面是否已签名？

---

## 二、材料检查

- [ ] 报销单是否填写完整？
- [ ] 经费卡号 / 项目号是否正确？
- [ ] 预算科目是否正确？（别走错科目！）
- [ ] 是否附上相关审批单？
- [ ] 是否附上会议通知 / 邀请函？（会议费）
- [ ] 是否附上行程单 / 登机牌？（差旅费）
- [ ] 是否附上采购申请单？（采购类）
- [ ] 是否附上合同？（如有）
- [ ] 所有材料是否按顺序整理好？

---

## 三、签字检查

- [ ] 导师是否已签字？
- [ ] 经手人是否已签字？
- [ ] 验收人是否已签字？（采购类）
- [ ] 是否需要学院负责人签字？

---

## 四、系统填报检查

- [ ] 系统报销单是否已提交？
- [ ] 系统金额是否与纸质一致？
- [ ] 系统审批流程是否正确选择？
- [ ] 是否打印系统生成的报销单？

---

## 五、最后检查

- [ ] 所有材料是否已保存电子版 / 扫描件？
- [ ] 是否有之前被退回的同类问题？→ 查 [[../../09_问题与复盘/问题库|问题库]]
- [ ] 是否在财务办公时间内？

---

## 六、带上这些再去

- 📁 所有纸质材料
- 🖊 笔（可能需要现场改）
- 📱 手机里有电子版备份
- 🪪 校园卡

---
